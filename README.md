## 训练模型代码运行
将该项目导入至PyCharm中，打开__init__.py文件，该文件为启动文件。  
### 选择模式
点击运行后，会出现选择模式的提示，可输入的模式分为"train"、"dev"和"test"三种模式。
### 选择训练轮数
选择模式后可以选择训练轮数。   
在"train"模式下，输入多少就训练几个epoch。若存在model.pt文件就会读取后继续训练，反之参考"模型存储"。  
在"dev"模式下，可获取当前模型在**验证集下**的准确率。  
### 模型存储
当OCR_Project文件夹下不存在model.pt文件时，使用"train"模式会在每轮epoch训练后将模型存储一次。  
**该模型存储的内容包括模型参数(键为"model")**
### test模式  
test模式下，会读取test文件夹中的数据，然后进行预测并输出一个testLabels.csv文件。用于最后的提交。
